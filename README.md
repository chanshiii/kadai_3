## ①課題内容（どんな作品か）
- グー・チョキ・パーのどれかを押したときにランダムに形成された値(グー・チョキ・パー)が出力される。

## ②工夫した点・こだわった点
- 画像を挿入したこと。
- タイマーの設置。
- 得点票の設置。
- 音楽の設置。

## ③難しかった点・次回トライしたいこと(又は機能)
- タイマーを作成する際に関数を使用したが、関数の呼び出し方法やイベントの指定方法がまだちゃんと理解できていないので、徐々に慣れていきたい。
- ブロック崩しゲームを動画を見ながら作成したが、何が必要なのかを瞬時に把握できていない。パドルから外れてボールが着地したらゲームオーバーになるようにしたい。
